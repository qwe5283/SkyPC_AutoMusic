# Sky PC Auto Music

Sky PC Auto Music 是一款用于在游戏 Sky: Children of the Light 中自动弹奏乐器的软件。

## 功能

- 能够读取 Sky Studio 制作的乐谱
- 支持暂停，进度控制，倍速控制
- 能够批量导入乐谱
- 能够合并相邻节拍相同音节，更好支持延音乐器
- (开发中...) 连接多设备同步播放状态增强合奏体验
- (开发中...) 显示歌词
